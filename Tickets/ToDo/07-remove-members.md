# 07 - Remove Members

## Description
Allow users to remove a member from the group by clicking a delete button next to their name.

## Tasks
- Create a DELETE `/api/members/:id` endpoint
- Add a delete button to each member in the list
- Send a delete request when the button is clicked
- Refresh the member list after deletion

## Done when
- Each member has a delete button
- Clicking delete removes the member from the database
- The list updates immediately after deletion
