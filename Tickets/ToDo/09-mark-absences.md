# 09 - Mark Absences

## Description
Allow users to mark members as absent for a specific session. Absent members should be skipped in the rotation for that session.

## Tasks
- Add a way to mark members as absent before recording a session
- Show checkboxes or toggles next to each member to set attendance
- Pass absence information when recording the session
- Skip absent members in the rotation logic

## Done when
- Members can be marked as absent for a session
- Absent members are skipped in the driving and payment rotation
- The correct next person is selected when someone is absent
