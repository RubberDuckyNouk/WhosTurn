# 07 - Track Payment Turns

## Description
Implement a rotation system to track whose turn it is to pay for drinks. All members participate in the payment rotation regardless of carpool status.

## Tasks
- Add a `pay_order` column or a separate tracking mechanism
- Write logic to determine whose turn it is to pay next
- Display the current payer prominently on the page
- Show the upcoming payment order

## Done when
- The app shows whose turn it is to pay
- All members are included in the payment rotation
- The rotation order is visible
